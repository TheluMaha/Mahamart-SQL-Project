# Mahamart SQL Project

## Project Overview
Mahamart is an E-Commerce Database Management System developed using MySQL. It manages customers, products, categories, orders, payments, and order details.

## Features
- Customer Management
- Product Management
- Order Management
- Payment Tracking
- Category Management
- Business Analysis Queries
